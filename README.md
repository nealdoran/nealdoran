## Dr. Neal A. Doran

**Geoscientist & Scientific Data Engineer**

I build deployed AI/ML systems and production-ready data pipelines optimized for massive scientific and biodiversity databases. Backed by a PhD in geological sciences (micropaleontology) and two decades of hands-on domain experience with deep-time geological, paleontological, and spatial data, I bridge the gap between highly complex natural-science datasets and robust computational engineering.

### Core Deployed Systems

- **Global Geology PostGIS Analysis** — A PostgreSQL/PostGIS spatial database integrating USGS World Geology and State Geologic Map Compilation datasets (148.3M km² of mapped geology), using spatial SQL to compute continental-scale geologic surface analysis. 👉 [View Project](https://github.com/nealdoran/global-geology-postgis)

- **Spatial Biodiversity Gap Audit Platform** — An automated extract, transform, load (ETL) pipeline and live analytical dashboard cross-auditing global IUCN Red List conservation data against a 26-million-record GBIF occurrence dataset using Python and SQLite. 👉 [Launch Deployed App](https://nealdoran-biodiversity-gap-audit.streamlit.app/)

- **Semantic Socratic Tutor** — A retrieval-augmented generation (RAG) system grounded in a specialized historical and philosophical scientific corpus. Built with Python, utilizing all-MiniLM-L6-v2 sentence-transformer embeddings, NumPy cosine-similarity retrieval, and an integrated Anthropic Claude API pipeline. 👉 [Launch Deployed App](https://socratic-tutor-ascznqxqjn6mrpzqc6gopp.streamlit.app/)
